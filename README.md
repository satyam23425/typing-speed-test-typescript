# ⌨️ Typing Speed Test – TypeScript

A web-based typing speed test application built using TypeScript that measures typing speed (WPM), accuracy, and time taken. This project demonstrates DOM manipulation, event handling, and real-time data tracking.

---

## 🚀 Features

- ⏱️ Real-time typing timer
- 📊 Words Per Minute (WPM) calculation
- 🎯 Accuracy tracking
- 🔄 Start / Stop / Reset functionality
- 📝 Dynamic sentence rendering

---

## 🧪 Testing & Validation

- ✅ Tested typing input handling and keystroke events
- ✅ Verified WPM calculation logic
- ✅ Validated accuracy computation
- ✅ Tested edge cases (empty input, reset, fast typing)
- ✅ Debugged timing and UI update issues
